# 座標が小数点以下第3位までの精度

- プロンプトでの改善は困難
- 道路外でも妥当な地点（公園・建物内等）の扱いを整理した上で，座標補正の手法を検討

# 道路に沿った距離が指定した時間を超える

- 撮影地点以外の移動経路も生成させるようにする

# 撮影者による被写体カテゴリの偏りがない

- プロンプトでエージェントに多様なペルソナを与えて興味を持たせるようにする
